Mindmap

介绍​

Mindmap 是一款强大而优雅的思维导图工具，完全基于 JavaScript 构建，为创意思考和知识整理提供了理想平台。与众多同类工具不同，它无需依赖后端服务器，所有数据和操作都在本地浏览器中完成，确保了你的隐私安全和离线可用性。

这款工具融合了直观的用户界面与丰富的功能，使得组织思想、规划项目、梳理知识体系变得轻松自然。无论你是学生、教育工作者、研究人员还是项目经理，Mindmap 都能帮助你更清晰地展现思维结构，提升思考效率。

核心功能​

  • 丰富的主题样式：内置多种精美主题，支持自定义样式，适应不同场景需求
  • 灵活的节点编辑：支持文本格式化、图标、超链接、备注、标签等丰富节点属性
  • 多样的布局模式：提供逻辑图、思维导图、组织结构图、目录组织图等多种布局
  • 便捷的操作体验：拖拽调整、快捷键支持、撤销/重做、节点自由移动
  • 全面的导入导出：支持 Markdown、Xmind、FreeMind 等格式导入，以及 PNG、SVG、PDF、JSON 等格式导出
  • 本地存储安全：数据存储在浏览器本地，保护隐私，支持手动备份
  • 协作分享功能：生成分享链接或图片，便于团队协作或成果展示
  • 完全离线可用：一旦加载完成，即使断网也能正常使用所有功能

部署​

Mindmap 提供了灵活的部署选项，满足不同需求：

快速部署​

  1. 克隆官方仓库：git clone https://github.com/wanglin2/mind-map.git
  2. 安装依赖：npm install
  3. 开发模式：npm run serve
  4. 构建生产版本：npm run build
  5. 将构建文件部署到任何静态网站托管服务（如 GitHub Pages、Vercel、Netlify 等）

Docker 部署​

docker run -d -p 8080:80 hraulein/mind-map:latest

更详细的部署指南、二次开发和 API 文档，请参考官方文档：https://wanglin2.github.io/mind-map-docs/

使用指南​

在线体验​

你可以直接访问我部署的实例快速体验：https://mindmap.vip.oneboy.app/

无需注册账号，打开即用，所有数据存储在你的浏览器中。

基本操作​

  1. 创建思维导图：点击"新建"按钮开始一个空白导图，或从模板开始
  2. 添加节点：
    • 选中节点后按 Tab 键添加子节点
    • 按 Enter 键添加同级节点
    • 使用右键菜单或工具栏添加各类节点
  3. 编辑内容：双击节点进行文本编辑，支持富文本格式
  4. 调整结构：拖拽节点调整位置，或使用快捷键调整层级
  5. 应用样式：通过右侧面板更改主题、颜色、字体等样式
  6. 保存与导出：支持自动保存到本地，以及多种格式导出分享

使用场景​

  • 学习笔记整理：将课程内容或读书笔记组织成结构化的知识地图
  • 项目规划：分解项目目标、任务和资源，创建清晰的路线图
  • 创意发想：通过头脑风暴捕捉和扩展创意，发现新的联系
  • 会议记录：实时记录会议内容，保持讨论的逻辑性和完整性
  • 知识管理：构建个人或团队的知识库，便于回顾和分享

Mindmap 的优势在于它完美平衡了功能丰富度和使用简便性，无需复杂设置即可创建专业级思维导图，同时保证了数据的私密性和可访问性。无论是日常工作还是学习规划，它都是你思维整理和创意展现的得力助手。
