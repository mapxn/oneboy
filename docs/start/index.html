开始

写在最前面，本分类记录一些自托管应用和有趣的小项目。在开始探索之前，我们需要做好以下基础准备：

邮箱​

Gmail​

谷歌邮箱是谷歌生态系统的基础账号。建议启用双因素认证(2FA)并配置密钥登录，同时绑定Outlook作为安全备用邮箱，提升账号安全性。

Outlook​

微软邮箱同样是微软账户体系的核心。它支持最多创建10个别名，相当于拥有11个邮箱地址，所有发送到这些地址的邮件都会统一收集到一个收件箱中。同样建议开启双因素认证和密钥登录以增强安全性。

为了防范社会工程学攻击，理想的做法是为每个网站服务使用独立的邮箱地址。市面上有多家服务商提供此类功能，例如DuckDuckGo的Duck.com服务。个人推荐的方案是通过自有域名建立专属邮局系统。

核心账户​

Github​

作为程序员的必备平台，GitHub不仅提供代码托管服务，更有众多开源项目和实用工具。即使非技术人员也能在此发现丰富的资源和免费服务，值得每个人注册使用。

Cloudflare​

被誉为互联网的大善人，Cloudflare提供全面的网络服务：域名注册、DNS托管、免费CDN、WAF防火墙、R2存储等。对于任何想要构建网络项目的人来说，这都是一个不可或缺的平台。

Vercel​

另一位互联网大善人，Vercel由Next.js创始团队打造，主要提供前端云平台服务。其便捷的部署流程和强大的托管能力，让前端项目和后端服务的管理变得异常简单。

域名​

选择一个好的域名是数字身份的第一步。推荐在国际平台注册域名，如Spaceship、Namecheap、Namesilo或Cloudflare等，以获得更好的服务和保障。

拥有个人域名后，你可以探索的可能性将会大大扩展，从个人网站到自托管服务，一切皆有可能。
